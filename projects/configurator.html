Project Overview

Angular-based configurator tool associated to wordpress website, it allows users to visualize Zillege products in real-life images. Users can choose custom colors for the products and click on a link to view the product page. The tool is fully responsive, interactive, and designed based on a modern Figma layout, providing a smooth and engaging user experience across desktop, tablet, and mobile devices.

Technical Summary

    • UI/UX Implementation: Developed an interactive Angular application replicating a modern e-commerce configurator with pixel-perfect precision and smooth navigation.
    • Product Customization: Enabled users to apply Zillege products on real-life images, choose custom colors, and view product pages via clickable links.
    • Responsive Design: Optimized layout for seamless performance across desktop, tablet, and mobile devices.
    • Deployment: Built a subdomain and successfully deployed the application on Hostinger, ensuring stable hosting and accessibility.
